# FE_TEMPLATE

使用 Bootstrap 的前端開發範本

v 1.1

## 環境需求

node 14 以上

## 安裝

git clone https://github.com/BLOCKCODE-TW/FE_TEMPLATE.git

remove .git

rename folder name

yarn install

修改 gulpfile.js 裡面的 projectname

gulp

## 開發注意事項

修改 src 下的

index.html

## 測試

ngrok http http://localhost:8080
